## 📋 Шаг 2: Разместите на Railway (бесплатно)

### 2.1 Регистрация на Railway

1. Перейдите на https://railway.app
2. Нажмите **Login** → **Login with GitHub**
3. Разрешите доступ

### 2.2 Создание проекта

1. Нажмите **New Project**
2. Выберите **Deploy from GitHub repo**
3. Найдите ваш репозиторий `ai-telegram-bot`
4. Нажмите **Deploy Now**

### 2.3 Настройка переменных

1. В панели проекта нажмите **Variables**
2. Добавьте переменные:

| Variable | Value |
|----------|-------|
| `TELEGRAM_TOKEN` | `8547446073:AAEngENBF0JU--W7j328G9QCwsw5G49OCWQ` |
| `GEMINI_API_KEY` | `AIzaSyAPtmCeedAupDyGnZedjKHrQbiID3aBCCI` |

3. Нажмите **Add**

### 2.4 Запуск

Railway автоматически запустит бота! 

Подождите 2-5 минут пока идёт сборка.

---

## 📋 Шаг 3: Проверка работы

### 3.1 Проверьте логи

1. В панели Railway → **Deployments**
2. Нажмите **View Logs**
3. Вы должны увидеть:
   ```
   ✅ Gemini модель gemini-2.0-flash-exp инициализирована
   🚀 Запуск бота (polling mode)...
   ```

### 3.2 Протестируйте бота

1. Откройте Telegram
2. Найдите вашего бота (по username который вы создали)
3. Отправьте `/start`
4. Бот должен ответить!

### 3.3 Если бот не отвечает

1. Проверьте логи на Railway
2. Убедитесь, что токен правильный
3. Проверьте, что бот не заблокирован

## 🎁 Альтернатива: Render.com (полностью бесплатно)

Если Railway требует карту, используйте Render:

1. Зарегистрируйтесь: https://render.com
2. New → **Web Service**
3. Подключите GitHub репозиторий
4. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python railway_bot.py`
5. Добавьте переменные окружения (как в Шаге 2.3)
6. Нажмите **Create Web Service**

## ❓ Частые проблемы

### "Bot is not found"
- Убедитесь, что токен правильный
- Проверьте, что бот существует через @BotFather

### "Gemini API error"
- Проверьте ключ GEMINI_API_KEY
- Убедитесь, что нет лишних пробелов

### Railway требует карту
- Это нормально
- $5 даётся сразу бесплатно
- Или используйте Render (не требует карту)

### Бот работает медленно
- Это нормально для бесплатных тарифов
- Gemini API может отвечать 2-5 секунд
