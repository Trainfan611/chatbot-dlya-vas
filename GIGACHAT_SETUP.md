# 🇷🇺 GigaChat от Сбера - настройка

## Почему GigaChat?

✅ **Работает в России** без прокси  
✅ **Бесплатно** для тестирования  
✅ **Русский язык** - родной для модели  
✅ **Быстро** - серверы в РФ  

---

## 📋 Шаг 1: Регистрация в Sber ID

1. Перейдите на https://developers.sber.ru/
2. Нажмите **Войти** → **Sber ID**
3. Войдите через:
   - Телефон (Сбербанк Онлайн)
   - Email
   - Госуслуги (опционально)

---

## 📋 Шаг 2: Создание приложения

1. После входа перейдите в **Личный кабинет**
2. Нажмите **Создать приложение**
3. Заполните:
   - **Название:** AI ChatBot (любое)
   - **Описание:** Telegram бот с AI
   - **Redirect URI:** `https://localhost`

4. Нажмите **Создать**

---

## 📋 Шаг 3: Получение токена GigaChat

1. В личном кабинете выберите ваше приложение
2. Перейдите в раздел **GigaChat**
3. Нажмите **Подключить**
4. Скопируйте **Auth Key** (Bearer токен)

Формат: `eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...`

---

## 📋 Шаг 4: Вставка токена в проект

Откройте файл `.env`:

```bash
notepad C:\Users\Dell\ai_chatbot_bot\.env
```

Замените:

```env
GIGACHAT_AUTH_KEY=ваш_токен_здесь
```

---

## 📋 Шаг 5: Проверка работы

Запустите тест:

```bash
cd C:\Users\Dell\ai_chatbot_bot
python test_gigachat.py
```

Если видите ответ - всё работает! ✅

## ❓ Частые проблемы

### "Неверный токен"
- Убедитесь, что скопировали весь токен
- Проверьте, что нет пробелов

### "Приложение не активно"
- Подождите 5-10 минут после создания
- Проверьте email подтверждения

### "Лимит превышен"
- Sandbox: 100K токенов/мес
- Создайте новое приложение
